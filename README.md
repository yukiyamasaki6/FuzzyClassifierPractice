# Fuzzy Classifier Practice

ファジィ論理分類器の練習プロジェクト

## 環境セットアップ

### 自動セットアップ（推奨）

リポジトリをクローンした後、以下のコマンドを実行してください：

**Linux/Mac:**

```bash
chmod +x setup.sh
./setup.sh
```

**Windows:**

```batch
./setup.bat
```

### 手動セットアップ

Python 依存関係のインストール：

```bash
pip install -r requirements.txt
```

## 必要な環境

- Python >= 3.8
- pip
- Visual Studio Code
- Python Extension for VSCode

## 依存関係

### Python

- numpy: 数値計算ライブラリ
- pandas: データ分析ライブラリ
- scikit-learn: 機械学習ライブラリ
- scikit-fuzzy: ファジィ論理ライブラリ
- matplotlib, seaborn, plotly: 可視化ライブラリ

## 使用方法

1. セットアップ完了後、VSCode でプロジェクトフォルダを開く
2. Python 環境が正しく設定されていることを確認
3. `.py`ファイルまたは`.ipynb`ファイルを作成して開発開始
